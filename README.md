# Little Light

A helper application for [Destiny 2](https://www.bungie.net/7/en/Destiny/NewLight).

## Status

Early development — project scaffolding is still being set up. Watch this space.

## Goals

- Make day-to-day Guardian life easier: quick access to the information you need between activities.
- Stay lightweight and fast — a companion, not a second game client.
